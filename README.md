# Albion House

Amber-text portfolio site for Albion House game projects.

The page is a single static `index.html` designed to feel like early-90s software running on an Apple IIe:

- dark background
- amber monochrome text
- simple text-first navigation
- minimal chrome and no computer bezel

The current content includes:

- an about section for the studio story
- a project catalog with `KILLBOX.BAS`
- a contact section
